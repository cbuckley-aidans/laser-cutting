Introduction

Learning Intentions and Success Criteria

Learning Intentions:

  1. Learn how to efficiently align objects both vertically and horizontally
  2. Master shape operations: Unite, Minus, and Exclude
  3. Learn how to use the measuring tool to get precise measurements
  4. Learn how to use the Shape Builder tool for intuitive shape creation
  5. Understand how to organize and manage content using layers
  6. Learn how to use the Offset Path feature to create outlines and borders
  7. Master creating offset paths for text using expand and compound shapes

Success Criteria:

  1. Successfully align multiple objects using Illustrator's alignment tools
  2. Successfully create complex shapes by combining simpler shapes using Unite, Minus, and Exclude operations
  3. Successfully measure objects and distances using the measuring tool
  4. Successfully use the Shape Builder tool to create and remove parts of shapes intuitively
  5. Successfully organize content by moving objects between layers using the layers panel
  6. Successfully create clean outline paths using the Offset Path feature
  7. Successfully create text outlines using expand, compound shapes, and offset path
